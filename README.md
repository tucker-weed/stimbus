# Stimbus

Adds global event bus functionality to stimulus controllers

- Hosted via npm
  - Choose install method
    - `yarn add stimbus`
    - `npm i stimbus`
  - Module page: https://www.npmjs.com/package/stimbus

## Quickstart

- Create build: `yarn build`
- Run tests: `yarn test`
- Run formatter: `yarn format`
- Run linter: `yarn lint`
